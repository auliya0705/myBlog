TIPE DATA C++

Umumnya tipe data digunakan untuk membuat variable yang nanti akan mewakili data dari macam-macam jenis tipe data di dalam pemrograman C++. Macam-macam data bisa berisi huruf, angka, karakter, dan lain-lain. Oleh karena itu, tipe data dibutuhkan dalam bahasa pemrograman. Berikut macam-macam jenis tipe data di dalam pemrograman.

INTEGER

Integer (int) adalah tipe data numerik berupa bilangan bulat.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 1
x merupakan nama_variabel
1 merupakan value_variabel

FLOAT

Float adalah tipe data numerik berupa bilangan real atau bilangan pecahan atau decimal.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 3.14
x merupakan nama_variabel
3.14 merupakan value_variabel

DOUBLE

Double adalah tipe data numerik berupa bilangan real atau bilangan pecahan atau decimal yang nilainya lebih banyak dari float.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 3.14782937
x merupakan nama_variabel
3.14782937 merupakan value_variabel

CHAR

Char (karakter) adalah tipe data berupa karakter tunggal yang menggunakan huruf, angka, dan simbol.
Format penulisan deklarasi adalah nama_variabel = 'value_variabel'.
Contoh :
nilaiku = 'A'
nilaiku merupakan nama_variabel
'A' merupakan value_variabel

STRING

String adalah tipe data berupa teks.
Format penulisan deklarasi adalah nama_variabel = "value_variabel".
Contoh :
namaku = "Luffy"
namaku merupakan nama_variabel
"Luffy" merupakan value_variabel
