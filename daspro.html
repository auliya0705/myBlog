TIPE DATA C++

Umumnya tipe data digunakan untuk membuat variable yang nanti akan mewakili data dari macam-macam jenis tipe data di dalam pemrograman C++. Macam-macam data bisa berisi huruf, angka, karakter, dan lain-lain. Oleh karena itu, tipe data dibutuhkan dalam bahasa pemrograman. Berikut macam-macam jenis tipe data di dalam pemrograman.

INTEGER

Integer (int) adalah tipe data numerik berupa bilangan bulat.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 1
x merupakan nama_variabel
1 merupakan value_variabel

FLOAT

Float adalah tipe data numerik berupa bilangan real atau bilangan pecahan atau decimal.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 3.14
x merupakan nama_variabel
3.14 merupakan value_variabel

DOUBLE

Double adalah tipe data numerik berupa bilangan real atau bilangan pecahan atau decimal yang nilainya lebih banyak dari float.
Format penulisan deklarasi adalah nama_variabel = value_variabel.
Contoh :
x = 3.14782937
x merupakan nama_variabel
3.14782937 merupakan value_variabel

CHAR

Char (karakter) adalah tipe data berupa karakter tunggal yang menggunakan huruf, angka, dan simbol.
Format penulisan deklarasi adalah nama_variabel = 'value_variabel'.
Contoh :
nilaiku = 'A'
nilaiku merupakan nama_variabel
'A' merupakan value_variabel

STRING

String adalah tipe data berupa teks.
Format penulisan deklarasi adalah nama_variabel = "value_variabel".
Contoh :
namaku = "Luffy"
namaku merupakan nama_variabel
"Luffy" merupakan value_variabel

BENTUK PERCABANGAN PEMROGRAMAN

Fungsi percabangan yaitu untuk memecahkan masalah dan mengambil keputusan di antara beberapa pilihan tergantung pada situasi yang dihadapi program. Terdapat beberapa bentuk percabangan pemrograman dalam C++, yaitu :

PERCABANGAN IF

Percabangan if merupakan percabangan yang hanya memiliki satu blok pilihan saat kondisi bernilai benar. Pada percabangan IF, jika kondisi terpenuhi akan memberikan hasil pernyataan dari IF tersebut, tetapi jika tidak terpenuhi maka tidak akan memberikan hasil apa-apa.
Penulisan percabangan IF yaitu :

if (kondisi) then 

        aksi

PERCABANGAN IF/ELSE

Ada suatu tambahan agar IF dapat berjalan sesuai keinginan kita yaitu ELSE, yang mempunyai fungsi memberikan hasil jika pernyataan IF tidak terpenuhi.
Penulisan percabangan IF yaitu :

if (kondisi) then 

        aksi-1
else (not kondisi) 
        aksi-2
Contoh :



Pada kasus tersebut apabila nilai inputan lebih dari 5, maka output yang keluar adalah
Berhasil!! Nilai lebih dari 5

Namun apabila nilai inputan kurang dari 5, maka output yang keluar adalah
Nilai kurang dari 5

PERCABANGAN IF/ELSE/IF

Else if berarti sebuah pilihan selain if else. Dibutuhkan jika ingin membuat pertanyaan lain lagi.

if (kondisi-1) then 

        aksi-1
else if (kondisi-2) then 
        aksi-2
else (not kondisi) 
        aksi-3
Contoh :



Pada kasus tersebut apabila nilai inputan lebih dari sama dengan 85, maka output yang keluar adalah
Kimia Anda A

Apabila nilai inputan lebih dari sama dengan 70, maka output yang keluar adalah
Kimia Anda B

Apabila nilai inputan lebih dari sama dengan 55, maka output yang keluar adalah
Kimia Anda C

Namun apabila nilai inputan tidak sesuai dengan ketiga kondisi tersebut, maka output yang keluar adalah
Kimia Anda D

PERCABANGAN BERSARANG (NESTED)

Nested IF adalah pernyataan IF yang ada di dalam pernyataan IF.

if (kondisi) then 

        aksi //bisa dibuat bisa tidak
        if (kondisi) then
                 aksi
        else
                 aksi
else 
        aksi
        if (kondisi) then
                 aksi
        else
                 aksi

SWITCH/CASE

Percabangan switch/case adalah bentuk lain dari percabangan if/else/if. Kondisi SWITCH CASE terdiri dari 2 bagian, yakni perintah SWITCH dimana terdapat nama variabel yang akan diperiksa, serta 1 atau lebih perintah CASE untuk setiap nilai yang akan diperiksa.
Penulisan percabangan SWITCH/CASE yaitu :

switch (variabel) 

    case (value):
             //blok kode
             break;
    case (value):
             //blok kode
             break;
    default:
             //blok kode
